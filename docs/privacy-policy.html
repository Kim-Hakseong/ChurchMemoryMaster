교회학교 암송 수첩 개인정보처리방침

시행일: 2026년 6월 3일

본 앱은 사용자의 개인정보를 외부 서버로 전송하거나 수집하지 않습니다. 모든 데이터는 사용자의 기기 내부에만 저장됩니다.

1. 수집하는 정보

본 앱은 다음 정보를 사용자 기기 내부에만 저장하며, 외부로 전송하지 않습니다.

  • 암송 진행 상태(체크 기록, 진도율, 포인트, 배지, 연속 일수)
  • 북마크한 말씀 목록
  • 사용자가 설정한 알림 시간 및 폰트 크기
  • 사용자가 직접 추가한 메모

위 정보는 모두 기기의 로컬 저장소(LocalStorage, IndexedDB)에 저장되며, 앱을 삭제하면 모두 함께 제거됩니다.

2. 수집하지 않는 정보

  • 이름, 이메일, 전화번호 등 개인 식별 정보
  • 위치 정보
  • 연락처 및 주소록
  • 사용 통계 또는 분석 데이터
  • 광고 식별자

3. 앱이 요청하는 권한

권한용도
인터넷앱 내부 웹뷰 동작에 필요 (외부 서버 통신 없음)
알림사용자가 설정한 시간에 암송 알림 표시 (모든 알림은 기기 내에서 발생)
사진/저장소암송 카드 이미지를 사용자 갤러리에 저장할 때만 사용
마이크 (Android)Capacitor 프레임워크 기본 권한이며, 실제로 녹음하지 않습니다

4. 제3자 제공

본 앱은 사용자의 어떠한 정보도 제3자에게 제공하지 않습니다.

5. 외부 서비스

본 앱은 광고, 분석 SDK, 소셜 로그인 등의 외부 서비스를 사용하지 않습니다.

6. 데이터 보관 및 삭제

모든 데이터는 사용자의 기기에만 저장되므로, 다음 방법으로 직접 삭제할 수 있습니다.

  • 앱 내 설정 메뉴에서 "데이터 초기화"
  • 운영체제의 앱 정보 → 저장공간 → 데이터 지우기
  • 앱을 삭제하면 모든 데이터가 함께 제거됩니다

7. 아동의 개인정보 보호

본 앱은 만 14세 미만 아동의 개인정보를 수집하지 않으며, 모든 기능이 개인정보 수집 없이 사용 가능합니다.

8. 보안

모든 데이터는 사용자 기기 내부에 저장되며, 외부 네트워크로 전송되지 않습니다. 따라서 통신 과정에서의 정보 유출 위험이 없습니다.

9. 개인정보처리방침 변경

본 방침이 변경되는 경우 앱 업데이트 또는 본 페이지를 통해 공지합니다.
